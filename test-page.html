StreamVolume Guard - Test Page

Cette page crée un vrai fichier audio WAV local dans un élément <audio>. Pour éviter les permissions file://, lance-la de préférence avec node tests/start-local-server.js.

Démo avant / après

Compare les mêmes niveaux bruts sans puis avec l'extension active. C'est le test le plus rapide pour vérifier l'intérêt avant un live.

Cette page crée trois niveaux précis : faible à -63.0 dB, fort à -43.0 dB, puis très fort à -3.0 dB. Le très fort est volontairement beaucoup plus haut que le fort pour que l'écart soit net à l'oreille.
